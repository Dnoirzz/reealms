# Suggested commands
## Windows shell basics
- `Get-ChildItem`
- `Get-ChildItem -Recurse`
- `rg pattern path`
- `Select-String -Path file -Pattern text`
- `cd <path>`
- `git status --short`
- `git diff -- <path>`

## Flutter app (`Reealms/`)
- `cd Reealms`
- `flutter pub get`
- `flutter run`
- `flutter test`
- `flutter analyze`
- `dart format lib/`
- `flutter build apk --release`

## Expo app (`ReealmsExpo/`)
- `cd ReealmsExpo`
- `npm install`
- `npm start`
- `npm start -- --tunnel -c`
- `npm test`
- `npm run typecheck`
- `npx expo-doctor`
- `npx expo export --platform android`
